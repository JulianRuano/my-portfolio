Movie API
Node.js JavaScript Express.js HTML CSS

Una API desarrollada con Node.js y Express.js para administrar información sobre películas, incluyendo la capacidad de crear, leer, actualizar y eliminar registros de películas.

  • Rutas para listar todas las películas y obtener detalles de una película específica.
  • Posibilidad de crear nuevas películas y actualizar información existente.
  • Eliminación de películas por su ID.
  • Patrón de diseño Modelo-Vista-Controlador (MVC) para una organización estructurada del código.
